Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995, now owned by Oracle Corporation. It is widely known for its principle of "write once, run anywhere," which means code written in Java can run on any device that has the Java Virtual Machine (JVM). This portability is possible because Java code is compiled into platform-independent bytecode. Java is statically typed, which means variable types are declared and checked at compile-time, making it a robust and secure language.

Java has a rich set of libraries and frameworks that make development easier, especially for building enterprise-level applications, Android apps, web-based tools, and more. Key components include the Java Standard Edition (SE), Java Enterprise Edition (EE), and Java Micro Edition (ME). Java SE provides the core functionality, such as data structures, networking, GUI development (Swing, JavaFX), and basic input/output. Popular frameworks like Spring, Hibernate, and Apache Maven have expanded Java’s utility in building scalable, maintainable software solutions.

Java is a versatile language where you can print output using the System.out.println() method. To declare an integer, you might use int number = 5;. A basic loop can be written using for(int i = 0; i < 5; i++). Methods in Java are defined using the public static void structure. To compile a file, the javac MyProgram.java command is used in the terminal.

  • System.out.println("Hello, World!"); — Prints a message to the console.
  • int number = 10; — Declares and initializes an integer variable.
  • for(int i = 0; i < 5; i++) { } — A simple for loop.
  • if(number > 0) { } — A basic if statement to check a condition.
  • public static void main(String[] args) — The entry point of a Java application.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It was introduced by the W3C in 1996 to separate content from design, allowing developers to apply styles such as colors, layouts, and fonts across multiple web pages consistently. CSS improves the maintainability and flexibility of web design by enabling centralized style control, and it plays a critical role in responsive and modern web development.

CSS provides powerful tools to enhance the visual layout of web content. It uses selectors to target HTML elements and apply styles such as color, font-size, and background-color. CSS can be applied inline, within a <style> block in the HTML file, or externally using a .css file. Modern CSS features like Flexbox, Grid, and media queries enable responsive and adaptive web designs across different screen sizes.

CSS is a styling language where you can change text color using color: blue;. To apply spacing around elements, you might use margin: 10px;. Layouts can be created with display: flex;. You can set the font style with font-family: Arial, sans-serif;. To link a CSS file to your HTML, include it with <link rel="stylesheet" href="styles.css"> in the head section.

  • color: blue; — Sets text color to blue.
  • margin: 10px; — Adds spacing around an element.
  • font-size: 16px; — Sets font size.
  • display: flex; — Applies flexbox layout.
  • background-color: lightgray; — Sets background color.

HTML

HTML is a standard markup language used to create web pages and web applications. It was first developed by Tim Berners-Lee in 1991 and is now maintained by the World Wide Web Consortium (W3C). HTML is known for its structure-defining capability, allowing developers to organize content with elements like headings, paragraphs, lists, and links. It is platform-independent and forms the backbone of every website, working alongside CSS and JavaScript to create interactive user experiences.

HTML offers a wide range of elements to design the structure and layout of web content. It includes semantic tags like <header>, <section>, and <footer> for organizing content meaningfully. HTML5, the latest version, introduces multimedia elements like <audio> and <video>, as well as APIs for offline storage, geolocation, and more. It works seamlessly with CSS for styling and JavaScript for interactivity, making it a key component of modern web development.

HTML is a markup language where you can display headings using the <h1>Hello</h1> tag. To define a paragraph, use <p>This is a paragraph.</p>. Links are created using <a href="url">Link</a>. Images can be embedded with the <img src="image.jpg" alt="Image"> tag. To run HTML code, simply save it as a .html file and open it in a web browser.

  • <h1>Welcome</h1> — A top-level heading.
  • <p>This is a paragraph.</p> — A paragraph of text.
  • <a href="https://example.com">Link</a> — A hyperlink.
  • <img src="image.jpg" alt="Description"> — Displays an image.
  • <ul><li>Item</li></ul> — An unordered list.

Python

Python is a high-level, interpreted programming language developed by Guido van Rossum and first released in 1991. It is widely known for its simplicity, readability, and dynamic typing, which makes it easy for beginners to learn and powerful for professionals to use. Python is cross-platform, meaning code written in Python can run on various operating systems without modification, provided the Python interpreter is installed. Its strong standard library and support for multiple paradigms make it a versatile and secure language.

Python has a rich set of libraries and frameworks that simplify development, especially for web applications, data science, automation, and more. Key tools include the Python Standard Library, as well as popular external libraries like NumPy, Pandas, Matplotlib, and frameworks like Django and Flask. These tools make Python ideal for building scalable, maintainable, and fast-to-develop applications in a wide range of domains.

Python is a versatile language where you can print output using the print("Hello, World!") function. To declare an integer, you might use number = 5. A basic loop can be written using for i in range(5):. Functions in Python are defined using the def my_function(): structure. To run a file, the python my_script.py command is used in the terminal.

  • print("Hello, World!") — Prints text to the console.
  • number = 5 — Declares and initializes a variable.
  • for i in range(5): — A simple for loop.
  • if number > 0: — An if statement to check a condition.
  • def greet(): — Defines a function named greet.

Java Script

JavaScript is a high-level, interpreted programming language widely used to make web pages interactive. It was created by Brendan Eich in 1995 and has since become a core technology of the web alongside HTML and CSS. JavaScript enables dynamic content updates, form validation, animations, and communication with servers without reloading the page. It runs directly in the browser, making it essential for client-side scripting.

JavaScript can manipulate HTML and CSS using the Document Object Model (DOM), allowing developers to create rich user interfaces. It supports event handling, asynchronous programming with promises and async/await, and object-oriented patterns. JavaScript libraries like jQuery and frameworks like React, Angular, and Vue have further expanded its power, enabling complex front-end applications. It can also be used on the server side through environments like Node.js.

JavaScript is a scripting language where you can print output to the console using console.log("Hello, World!");. To declare a variable, you might write let count = 0;. A function can be defined using function greet() { }. Loops are written using syntax like for(let i = 0; i < 5; i++). You can link a JavaScript file to HTML with <script src="script.js"></script>.

  • console.log("Hello, World!"); — Logs output to the console.
  • let x = 10; — Declares a variable.
  • if (x > 5) { } — A simple if condition.
  • function greet() { } — Defines a function.
  • for (let i = 0; i < 5; i++) { } — A for loop.
